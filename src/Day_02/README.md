**Day 02 – Java Basics Practice Programs**

This day covers foundational Java programs focusing on arithmetic operations, conditionals, user input handling, and basic computational logic.
Each program includes input handling, validation (where needed), and correct formula application.

1. Simple Interest Calculator

**Description:**
Takes Principal, Rate (%), and Time (years) as input and calculates Simple Interest.

**Formula:**
Simple Interest (SI) = (P × R × T) / 100

**File Name:**
SimpleInterestCalculator.java


2. Area and Perimeter of a Rectangle

**Description:**
Accepts length and breadth and calculates the rectangle’s area and perimeter.

**Formulas:**
Area = length × breadth  
Perimeter = 2 × (length + breadth)

**File Name:**
RectangleAreaPerimeter.java

3. Swap Two Numbers Without Using a Third Variable

**Description:**
Takes two integers and swaps them using the arithmetic method.

**Logic:**
num1 = num1 + num2  
num2 = num1 - num2  
num1 = num1 - num2

**File Name:**
SwapTwoNumbers.java

4. Number Validation – Positive, Negative, or Zero

**Description:**
Reads a number and determines whether it is positive, negative, or zero.

**Logic:**

If n > 0 → Positive
If n < 0 → Negative
Else → Zero

**File Name:**
NumberValidation.java

5. Temperature Converter (Celsius ↔ Fahrenheit)

**Description:**
Converts Celsius to Fahrenheit and Fahrenheit to Celsius.

**Formulas:**
F = (C × 9/5) + 32  
C = (F − 32) × 5/9

**File Name:**
TemperatureConverter.java

6. Salary Calculator (Basic + DA + HRA)

**Description:**
Reads basic salary and computes total salary including:

DA = 40% of basic
HRA = 20% of basic

**Formula:**
Gross Salary = Basic + DA + HRA

File Name:
SalaryCalculator.java

7. Convert Days to Years, Weeks, and Days

**Description:**
Converts a given number of days into years, weeks, and remaining days.

**Conversion Logic:**

Years  = totalDays / 365  
RemainingDays = totalDays % 365  
Weeks = RemainingDays / 7  
Days  = RemainingDays % 7

**File Name:**
DaysConverter.java
